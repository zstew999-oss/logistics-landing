# Logistics Landing (React + TypeScript)

Commercial landing page for a logistics forwarding company focused on reliability, shipment control, and closing documents (B2B).

## Live Demo
https://b2b-logistics-landing.netlify.app/

## About
This project is a business-oriented landing page, not a tutorial or educational example.

The focus is on B2B logistics:
- responsibility for shipment
- control during transportation
- correct closing documents

## Tech Stack
- React
- Vite
- TypeScript
- CSS Modules

## Run locally
npm install
npm run dev
